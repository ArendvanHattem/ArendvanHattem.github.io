Project Unity

📁 View project on GitHub →
🎮 Play in browser →


Dit project heb ik in mijn tweede leerjaar software development gemaakt.

Het is een 3D spel gemaakt met Unity waarin je een blauwe bal bestuurt om gele kubussen op te pakken. Hierbij moet je omhoog, onder dingen door en zelfs springen.

Er is een rode enemy die je instakillt zodra hij je pakt. Ook is er een cube-counter die telt hoeveel kubussen je al hebt verzameld en een pauzeer en restart knop.

Je moet goed gebruik maken van je topsnelheid als je hem voor wil blijven,
want hij heeft onmiddellijke acceleratie en oneindige kracht om door de groene obstakels heen te duwen die jou vertragen.

Om alle kubussen te verzamelen zul je moeten plannen en snel moeten zijn. onder de tafel kan hij niet komen, daar zit je veilig.

Zorg dat je alle andere kubussen hebt verzameld voor je de sprong naar de laatste maakt, je kan niet meer terug.

Je kan bewegen met WASD of de pijltjestoetsen en als je op de pauzeer/speel knop hebt geklikt, kan je met ENTER pauzeren/spelen. Veel plezier met testen!
